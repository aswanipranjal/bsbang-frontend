bsBang - about

  • Extremely alpha prototype search using Bioschemas structured data.
  • Currently only indexes data from a hackishly pared down installation of InterMine containing only E. coli str. K-12 substr. MG1655 genes, but more to come from this and other sources.
  • Go to Github for the Solr config, the bsBang crawler and Solr setup code and this frontend. Patches and bug reports welcome.
  • Otherwise, please send suggestions/comments to justincc AT intermine.org or tweet me.
  • Brought to you by the Micklem Lab.
